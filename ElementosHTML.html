Elementos en linea y en bloque

En esta sección vamos a conocer algunos de los elementos HTML denomidanos en bloque y en linea:

Elementos en bloque

  • Section: representa una sección de la página web
  • li: se crean elementos en una lista
  • nav: Contiene elementos
  • output: formulario de salida
  • hgroup: es el encabezado de una sección

Elementos en linea

  • var: Es usada para una expresion matematica
  • small: letra pequeña
  • sup: sirve para elevar el texto como una potencia
  • abbr: ayuda a describir una abreviación
  • time: especifica un determinado tiempo
